# gedcom2html, a python script to convert Gedcom files to html files using bootstrap and jquery.
##Usage
```
gedcom2html myfile.ged
```
##Examples
[Dutch Royal Family](http://www.usekees.com/gedcom2.html/koninklijkhuis/
